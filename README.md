# 치지직 하이라이트 분석기

치지직(CHZZK) VOD 페이지에서 **채팅 밀도를 실시간 분석**하여 하이라이트 구간을 시각화해주는 크롬 확장프로그램입니다.

## ✨ 주요 기능

- **전체 채팅 밀도 분석**: 모든 채팅이 집중되는 구간을 탐지
- **10초 슬라이딩 윈도우**: 가장 채팅이 집중된 10초 구간을 기준으로 피크 탐지
- **60초 클립 자동 산출**: 피크 구간 전 25초 ~ 후 25초 = 총 60초 하이라이트 클립
- **드래그 가능한 플로팅 패널**: 영상 시청과 동시에 하이라이트 목록 확인 가능
- **타임스탬프 버튼 클릭**: 클릭 한 번으로 해당 시간대로 영상 즉시 이동
- **채팅 밀도 그래프**: Canvas 기반 시각화 차트
- **On/Off 토글**: 확장프로그램 아이콘 팝업에서 기능 즉시 켜기/끄기

## 🚀 설치 방법

1. 이 리포지토리 우측상단 초록색 'code' 버튼 누르고 ZIP으로 다운로드하거나 `git clone`
2. 다운로드한 zip 파일 압축해제
3. 크롬 브라우저에서 `chrome://extensions/` 접속
4. 우측 상단 **개발자 모드** 토글 켜기
5. **압축해제된 확장 프로그램을 로드합니다** 클릭
6. 다운로드한 `chzzk_highlight_ext` 폴더 선택

## 🎮 사용 방법

1. 치지직 다시보기(VOD) 페이지로 이동
2. 화면 우하단의 **⚡ 하이라이트 분석** 버튼 클릭
3. 채팅 데이터 로딩 완료 후 플로팅 패널에서 결과 확인
4. 타임스탬프 버튼 클릭 → 해당 구간으로 영상 이동

## ⚙️ 설정

확장프로그램 아이콘 클릭 시 팝업에서 설정 가능:

| 설정 | 기본값 | 범위 |
|---|---|---|
| 기능 On/Off | ON | - |
| 추출할 하이라이트 개수 | 20개 | 1 ~ 50개 |

## 🛠️ 동작 원리

```
전체 채팅 수집 (Pagination API)
    ↓
모든 채팅 타임스탬프 추출
    ↓
10초 슬라이딩 윈도우로 각 구간 채팅 밀도 계산
    ↓
밀도 내림차순 정렬 + 최소 간격 조건으로 피크 N개 선별
    ↓
각 피크 기준: clipStart = peakStart - 25초, clipEnd = peakEnd + 25초
    ↓
결과: 60초 하이라이트 클립 목록 + Canvas 그래프 시각화
```
